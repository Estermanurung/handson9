**Implementasi Abstrksi dalam Java
-ada 2 cara yaitu abstract class dan interface.

**INTERFACE
-ini adalah kontrak murni yang mendefinisikan oleh java class
yang menggunaan tanpa memberikan implementasi sama sekali.

**MULTIPLE INHERITANCE THROUGH INTERFACES
-Ini adalah salah satu keunggulan terbesar interface karna kemampuan
yang mampu memfasilitasi multiple inheritance.dan ini mampu 
mewarisi lebih dari saru parents. tidak bisa dari extend tapi 
mengizinkannya melalui interface.

**POLTYMORPHISM (Polimorfisme)
-ini berarti kemampuan suatu hal untuk memiliki banyak 
bentuk. memungkinkan kita menggunakan satu nama method yang sama.
-memiliki 2 cara utama:
1.Method Overriiding dan 
2.Method Overloading
